# ПРОЕКТ НЕ ЗАВЕРШЕН!
### Rubber End — нейросеть для сбора текстовых данных и расчёта срока годности шин.

### Как работает
📥 Загрузите текстовый документ .txt c условиями эксплуатации шин<br>
📤 Получите расчёт срока служб шин.

Цель проекта - определять точное время, когда шины приходят в негодность, чтобы снизить расходы на создание новых шин и улучшить экологию.
